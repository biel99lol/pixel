🎨 Conversor Imagem → Tabela Lua (Roblox)

Converte qualquer imagem para o formato de tabela que o Roblox entende (RGB + Alpha)

Resolução:
📊 Dimensões: - 🎨 Pixels: - 🌈 Cores únicas: -
📷 Imagem Original
🎨 Pixel Art (Prévia)
📝 Código Lua para Roblox (Formato Tabela)
🎮 Como usar no Roblox Studio:

1. Copie o código gerado acima
2. No Roblox Studio, crie um ModuleScript dentro de ServerStorage ou ReplicatedStorage
3. Cole o código no ModuleScript
4. Use o código em qualquer Script assim:

local pixelData = require(script.Parent.PixelArt) -- Seu ModuleScript

-- Criar GUI com os pixels
local screenGui = Instance.new("ScreenGui")
screenGui.Parent = game.Players.LocalPlayer:WaitForChild("PlayerGui")

for x, col in pairs(pixelData) do
    for y, pixel in pairs(col) do
        local r, g, b, a = unpack(pixel)
        if a > 0 then
            local frame = Instance.new("Frame")
            frame.Size = UDim2.new(0, 4, 0, 4)
            frame.Position = UDim2.new(0, x*4, 0, y*4)
            frame.BackgroundColor3 = Color3.fromRGB(r, g, b)
            frame.Parent = screenGui
        end
    end
end
