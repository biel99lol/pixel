📝 Código Lua para Roblox
🎮 Como usar no Roblox Studio:
1. Cole o código acima em um Script dentro de um ScreenGui
2. Execute o script para gerar a pixel art colorida automaticamente
3. O script criará um Frame com blocos de cores para cada pixel
4. Cada bloco é um TextLabel com cor específica e caractere definido
5. A arte será recriada perfeitamente com todas as cores originais
✨ Dica: Para melhor performance em imagens grandes, use qualidade Média (32x32)
