📝 Código Pixel Art para Roblox Cole no campo Text do TextLabel
🎨 Dica: O Roblox exibirá os blocos coloridos conforme as cores definidas no TextLabel!
🎮 Como usar no Roblox Studio:
1. Crie uma ScreenGui dentro do StarterGui
2. Adicione um TextLabel
3. Configure no TextLabel:
  • Font = "Code" ou "RobotoMono"
  • TextScaled = false
  • TextSize = 8-12 (ajuste conforme necessidade)
  • Text = cole o código gerado
4. Ajuste o tamanho do TextLabel para caber a arte (largura x altura)
5. Defina BackgroundTransparency = 1
6. TextColor3 = qualquer cor desejada (branco = 255,255,255)
✨ Dica extra: Para pixel art colorida, use RichText ou múltiplos TextLabels!
