📝 Código para Roblox Cole no TextLabel.Text
🎮 Como usar no Roblox:
1. Crie uma ScreenGui dentro do StarterGui
2. Adicione um TextLabel
3. Configure a fonte: Font = "Code" ou "RobotoMono"
4. Ajuste: TextScaled = false e TextSize = 10
5. Cole o texto gerado no campo Text
6. Ajuste o tamanho do TextLabel para caber a arte
7. Defina BackgroundTransparency = 1 e TextColor3 = 255,255,255
